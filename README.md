# TampermonkeyPlugins

# 油猴插件

## Gemini 

Gemini 官网自动加提示词

## csdn

打开文章后自动调用deepseek-r1:1.5b评论该文章，需要结合本地部署的ollama

## 抖音（douyin）

直播间评论实时采集，支持30秒内数据去重，每0.1秒检查一次新数据，需要本地先启动server_dy.py 才能正常使用服务器保存功能

## save_as_pdf

能够将何网页借助系统自带的打印能力，保存成pdf

## 小红书（xhs）

小红书文章采集器，需要先启动 server_xhs.py ，然后打开任何小红书文章，视频，卡片页面都能将页面数据保存成json，发送到服务器保存

## 云展网（yunzhan365）

自动采集云展网高清图片保存成pdf
